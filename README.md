# レスキュークローゼット (PWA)

奮発した服の死蔵を「着られた自己肯定感」に変えるサービスのUIプロトタイプ。

## ローカルで動かす
```
npm install
npm run dev
```
表示されたURLを開く。

## 公開（Vercel）
GitHubにこのフォルダを上げ、Vercelで「Import Project」→ フレームワークは自動で **Vite** が選ばれます。そのままDeployでOK（設定変更不要）。

## PWA（ホーム画面に追加）
公開後のURLをスマホのブラウザで開き、共有メニューから「ホーム画面に追加」。
全画面のアプリとして起動します。
